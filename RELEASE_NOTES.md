# Release Notes - v1.1.0

## 概要
設定メニューが OmniDesk のメインブラウザ画面（Webview）の背後に隠れてしまい、操作できない不具合を修正しました。
レイアウトを並列配置（サイドパネル形式）に変更することで、表示の競合を根本的に解決し、OmniDesk の操作性を向上させました。

## 新機能と改善
- **デザインの一新**: 設定パネルが右側にスマートにスライドインし、メイン画面を「押し出す」ような挙動になります。
- **Webview の追従リサイズ**: 設定メニューを開くと、開いているウェブページも自動的に縮小され、重なりが発生しません。
- **アニメーション**: 滑らかなフェードインとスライドイン効果により、モダンで洗練されたフィーリングを提供します。
- **内部安定性の向上**: サービス切り替え時のエラーハンドリングを強化し、不意の不具合に対する耐性を高めました。

## 技術的な変更点
- `AppContext` によるグローバルな設定パネル状態管理の導入。
- `App.tsx` における Flexbox ベースの 3 カラムレイアウト（Sidebar | Main | Settings）への移行。
- `useWebview` フックにおけるアクティブサービスIDのロールバックロジック修正。
- プレミアムな「桜ピンク」と「ネオンブルー」のカラーパレットに基づいたスタイリングの適用。
